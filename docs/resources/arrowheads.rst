Wind arrow heads
======================

The wind arrow head style can be controlled by the follwing parameters in :func:`mwind`:

- ``wind_arrow_head_shape``: possible values: 0, 1, 2, 3
- ``wind_arrow_head_ratio``: floating point number in the range of [0,1]
