dialog
=================

.. py:function:: dialog(widgets)
   
    *New in metview-python version 1.9.0*.
   
    Creates a popup editor from a list of widget components and returns user input.
      
    :param widgets: list of widget components
    :rtype: dict

    The example below shows how :func:`dialog` can be used with various widget components:

        .. code-block:: python

            import metview as mv

            a = mv.ui_icon(
                name="input data",
                class_="GRIB"
            )

            b = mv.ui_any(
                name="verif date (yyyy-mm-dd)",
                default="2012-03-01",
                help="help_script",
                help_script_command="echo Dates must be in YYYY-MM-DD format"
            )

            c = mv.ui_slider(
                name="days",
                min=1,
                max=10,
                default=5
            )

            d = mv.ui_option_menu(
                name="parameter", 
                values=["UV", "t"],
                default="t"
            )
        
            e = mv.ui_any(
                name="area",
                help="help_input",
                input_type="area",
                default=[30,-25,50,65] # S, W, N, E
            )

            # retrieve the user input values
            x = mv.dialog([a, b, c, d, e])

            print("area=", x["area"])
